# 会议室预定

[TOC]

## 1.项目需求

1.用户登录

2.用户进行会议室预定，可以在每天的早八点到晚6点的进行以小时为单位的预定

3.只可以预定当前时间点以后的会议室，

4.预定成功后，页面应该显示预定人名，以及将预定状态的会议室改为，蓝色，本人预定改为黄色

5.可以退订会议室

6.当天同一时间段，一个会议室只可以被一个人预定
